# News app
<p>
  This app displays the user the latest news from various sources with news caching inside the room.
  <br/>
  
  <br/><b>MVVM pattern</b>,
  <br/><b>Room</b>,
  <br/><b>Retrofit</b>,
  <br/><b>kotlin coroutines</b>,  
  <br/><b>Repository pattern</b>, and
  <br/><b>Dependency injection</b> are used.
</p>
